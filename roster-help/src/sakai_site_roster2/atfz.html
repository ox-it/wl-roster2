Changing roster permissions

Note: Only roles having the necessary authorization to change permissions are able to access this feature. To determine your role, follow the directions in Participant roles. For a basic understanding of roles and permissions, see Permissions and roles: Overview.

Roster permissions can be set on a site-by-site basis, to enable and disable certain features of the roster. For example, students cannot by default view all members of a site, but this restriction can be lifted on the permissions page by checking the roster.viewallmembers permission for the Student role.
